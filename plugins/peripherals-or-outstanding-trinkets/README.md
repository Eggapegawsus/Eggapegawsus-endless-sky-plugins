# Peripherals Or Outstanding Trinkets. 
<br>
Peripherals Or Outstanding Trinkets is a vanilla plus content expansion/quality of life 
overhaul plugin for Endless Sky. The plugin of itself isn't intended to be ridiculously overpowered, but it will likely give the player a decent advantage (At least when compared to heavily balanced-centered plugins, that is)

As of right now, it only contains additional outfits, but will have many more features in the upcoming releases
<br>

# Current Release Status

<details><summary>POOT Alpha Status</summary>

As it stands right now, POOT is a massive weapons and outfits expansion, and is in its alpha release. I consider it an alpha release as I intend to include FAR more than it currently contains, as it will likely have quite a few more missions, outfit/ships, gameplay features/mechanics, storylines, and general other content, as well as implementing better methods of encountering/obtaining currently included outfits. The majority of currently included content _should_ be fully working and able to be used in a normal playthrough (Albeit probably a little bit unbalanced as it currently stands), with a slim amount of outfits currently being unobtainable through normal gameplay.

 But right now, POOTs primary reason for release is for general input on balancing changes, adjustments, and fixing any found bugs before the larger content gets added. That being said, if you are using this plugin in its current Alpha state, please:

 - If there is a feature, balance adjustment, nerf or buff, or anything else you might like to see added, removed, or adjusted, please feel free to message/ping me on the Endless Sky discord, or post an issue with anything you might like to see included!

 Right now most outfits and weapons are availible (at least temporarily) through outfitters, with some set to be unlocked as the story progresses (I.E. when Electron Beam is unlocked, Flamethrower unlocked, Stack core unlocked, etc.). Atleast a few of these (in addition to everything within "TEMP-SALES.txt") will likely be removed from outfitters as more content/features get added, and they can be encountered as they are intended to be in the future. 
  - *NOTE: If an outfit or weapon is not availible anywhere, this plugin has built in Senza Solaria support, so Senza Solaria's dev start can be used to obtain and try all outfits, including unobtainable ones.*

</details>

<details><summary>Reactor Core Expansions (Click Me)</summary>

<blockquote>

Reactor Core Expansions are a new type of outfit that can be installed onto any ship with a power generating reactor that has capacity for them. When installed, they provide additional stat bonuses acting as a sort of "mini outfit", while also taking up no additional storage or outfit space of the ship they're installed on. Many of them, however, do have side effects (I.E. consume energy, generate heat, affect shield generation, lots of mass, ETC.)

Most reactors or energy generation outfits from the base game and from POOT should have capacity for atleast one, with the exception being very small or unique reactors. A select few other outfits may also have capacity for them.
